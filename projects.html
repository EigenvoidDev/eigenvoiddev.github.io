An online asymmetric multiplayer survival horror video game developed and published by Canadian studio Behaviour Interactive.

Last Updated: August 7, 2025

A command-line tool that can decrypt and re-encrypt data served from Dead by Daylight's private API. It handles both asset and profile data formats and supports branch-specific keys for QA, Stage, Cert, PTB, and Live environments.

Last Updated: August 8, 2025

A desktop GUI tool that allows you to control which AWS regions Dead by Daylight can connect to. It provides real-time latency and packet loss metrics for all supported regions and offers a simple interface for blocking or unblocking access to specific ones through hosts file rules.
